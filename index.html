  • Util, a class that has a bunch of one-off functions, including:
    • debounce(fn, ms) - bog standard debounce function.
    • getData(element, name) - using an internal WeakMap, gets arbitrary data stored by key.
    • setData(element, name, value) - using an internal WeakMap, stores arbitrary data (value) by key (name).
    • getAncestor(element, selector) - this is the same as element.closest(selector). If that doesn't exist, uses matches and parentElement to walk the tree and find the ancestor.
